EAUI（Easy Ad UI）是一个专为广告设计而优化的UI组件库，基于 TDesign ，使用 `Vue3` 、`TypeScript`、`Vite` 开发，旨在提供一套简单、高效且灵活的组件，帮助开发者快速构建高质量的广告界面。

# 📦 安装

```shell
npm i eaui
```
